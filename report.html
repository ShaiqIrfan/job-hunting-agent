Final Submission Report

This report documents the final submission for the Job Hunting Agent project. It is intended for printing as PDF from a browser.

GitHub Repository

https://github.com/ShaiqIrfan/job-hunting-agent

Project Checklist

  • GitHub repository link submitted
  • `input_jobs/` contains at least one job poster or JD file
  • `input_resumes/` contains at least one resume file
  • `input_kb/` contains at least one knowledge base file
  • Agent reads files from folders rather than hard-coded text
  • `outputs/` contains generated reports
  • `tracker/applications.csv` contains application status records
  • `tracker/reminders.txt` contains interview/follow-up reminders
  • `README.md` explains how to run the agent
  • `reflection.md` explains what was built, tested, and improved

Submission Contents

Folder / File Description
app.py Main Python application that reads input files, analyzes skills, generates reports, and writes outputs.
input_jobs/job_poster_01.txt Sample junior AI engineer intern job description used by the agent.
input_resumes/my_resume.txt Sample student resume formatted for beginner-level AI / software internship roles.
input_kb/kb_notes.txt Knowledge base notes covering machine learning and software engineering concepts.
outputs/job_analysis_report.txt Generated job analysis report based on the sample job description and extracted skills.
outputs/skill_gap_report.txt Generated report comparing job skills to resume skills and calculating match score.
outputs/tailored_resume_suggestions.txt Generated resume suggestions report highlighting missing skills.
outputs/interview_questions.txt Generated interview questions based on job skills, HR prompts, and knowledge base notes.
outputs/final_agent_report.txt Combined agent report including analysis, gap review, suggestions, and interview preparation.
tracker/applications.csv Application tracker with status records for sample applications.
tracker/reminders.txt Generated reminders for interview follow-up and application actions.
reflection.md Project reflection describing build, testing, and improvements.
